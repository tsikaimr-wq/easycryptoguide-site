# 项目功能说明

## 1. 这是什么项目

这是一个交易所风格的前端项目，包含：

- 官网展示与活动内容（`index.html`）
- 账号体系（注册/登录）
- 移动端交易主页面（`mobile.html`）
- 行情页（`markets.html`）
- 永续与交割合约交易页面
- 后台管理页面（`admin.html`）

## 2. 当前已实现功能（重点）

### 2.1 注册与登录

- 邮箱/手机号注册带唯一性校验，避免重复注册。
- 自动生成账号（`EC` + uid 前 8 位）。
- 支持“账号 + 密码”登录。

### 2.2 KYC 门禁

- 用户注册/登录后需通过 KYC 才能进入关键交易流程。
- 未 KYC 用户会被引导到认证流程，并拦截关键操作。

### 2.3 永续交易

- 支持开仓参数：`Market/Limit/Post Only`、杠杆、数量、TP/SL。
- 支持平仓结果展示：开仓价、平仓价、手续费、ROE、已实现盈亏。
- 支持历史订单记录展示与回看。
- 后台可通过用户结束价控制该用户平仓盈亏结果（P/L）。

### 2.4 交割合约

- 保持原有交割合约逻辑，并叠加必要门禁控制。

### 2.5 行情与资产页面

- 行情页与移动端币种列表已支持图标映射与加载失败回退。
- 资产页面支持折叠/展开，界面交互已做基础优化。

### 2.6 Favicon 与社交分享预览

- 已配置浏览器标签图标（favicon）。
- 已配置 Open Graph / Twitter Card，可用于 WhatsApp / Telegram 分享预览。
- 预览图资源：`assets/brand/og-preview.png`

## 3. 已知前置要求

- Firebase 域名白名单需配置为线上实际域名。
- 页面里的 `YOUR_DOMAIN` 占位符需替换后再发布。
